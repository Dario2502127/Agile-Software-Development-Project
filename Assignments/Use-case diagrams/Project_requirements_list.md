# The project requirements list
### Customer

* Search available rooms
* View room details (price, facilities)
* Make a booking
* Cancel a booking
* View booking status
* Make payment

### Receptionist

* Confirm customer bookings
* Update booking details (dates, room type, etc.)
* Cancel bookings on behalf of customer
* Manage check-in and check-out
* View booking history

### Admin

* Manage user accounts (customers, receptionists)
* Add / update / remove rooms
* Generate reports (bookings, revenue, occupancy)
* Configure system settings
* Monitor overall system activity
